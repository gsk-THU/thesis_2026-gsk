# 2月
本月重点完成测试体系的构建与系统架构设计，具体包括设计测试流程并编写测试代码、跑通测试验证硬件冗余性、设计实现多进程并行NPU驱动所需的功能模块并明确各模块职责、实现NPU任务状态监控功能。
# 3月
本月主要完成多进程支持机制的开发，包括实现内存隔离功能与互斥锁机制、NPU任务切换功能、NPU任务调度功能，并根据实际进度尝试实现更高级的调度算法以提升系统吞吐量。
# 4月
本月以实验验证与中期汇报为核心，主要完成中期报告的准备与汇报、实验方案设计、实验代码编写、多进程NPU驱动吞吐量测试及性能调优工作，全面验证驱动性能提升效果。
# 5月
本月主要完成实验结果的总结分析、技术报告撰写、毕业设计论文撰写及结题答辩准备工作，形成完整的研究成果文档。
